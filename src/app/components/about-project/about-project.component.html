Stock Tracker App

This project is developed as a part of the angular certification coding challenge from Angular Training
This is level 2 challenge to develop the coding application. This is a stock tracking application which uses open source api finnhub Finnhub to get real time stock data of American stock exchange.
The application initially fetches data from local storage which is saved from the last time user has used this application.
The application allows the user to add the stock to the list to keep its track, it also performs some basic validation on the input field.
The application uses different API end points to get different details about the stocks.
User once enters the symbol of the stock he wants to track and clicks on "Track Stock" button, the application calls 2 API
1) Finnhub API to fetch stock details like opening price, high price, percentage change, current price, low price from here
2) Finnhub API to fetch company stock details
With the following information new stock is added to the tracking list.
The application also provides ability to add stock and remove the stock from the list whenever needed with the close button present on each tracker.
The stock tracker has an additional feature to give sentiments of the stock for the past few months for which it uses open source. Finnhub API to get the sentiments of the stock for previous month from here. By default it calls for the past three months, but depending upon the API response the number months shown can be less
. The application is built with Finnhub Open source API, Angular and Angular Material library and Bootstrap.
